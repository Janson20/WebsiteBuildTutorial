# 域名购买指南

本教程介绍如何选择、购买和管理域名。

## 一、域名基础知识

### 1.1 域名结构

```
https://www.example.com
        │   │       │
        │   │       └── 顶级域(TLD)
        │   └────────── 二级域(SLD)
        └────────────── 子域(Subdomain)
```

### 1.2 顶级域名类型

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| 通用顶级域(gTLD) | .com, .net, .org | 通用网站 |
| 国家代码域(ccTLD) | .cn, .jp, .us | 特定国家/地区 |
| 新通用顶级域(new gTLD) | .dev, .app, .blog | 特定行业/用途 |
| 中文域名 | .中国, .公司 | 中文用户 |

### 1.3 域名选择建议

1. **简短易记**：控制在15个字符以内
2. **易于拼写**：避免容易拼错的单词
3. **品牌相关**：与品牌名或业务相关
4. **避免连字符**：容易混淆和拼写错误
5. **考虑SEO**：包含关键词有助于搜索排名

## 二、域名注册商选择

### 2.1 国内注册商

| 注册商 | 优势 | 注意事项 |
|--------|------|---------|
| 阿里云(万网) | 价格透明，管理方便 | 需要实名认证 |
| 腾讯云 | 与云服务集成 | 需要实名认证 |
| 西部数码 | 老牌注册商 | 界面较旧 |

### 2.2 国际注册商

| 注册商 | 优势 | 注意事项 |
|--------|------|---------|
| Namecheap | 价格实惠，免费隐私保护 | 支付方式有限 |
| Cloudflare | 原价注册，无溢价 | 需使用其DNS |
| NameSilo | 低价，免费隐私保护 | 界面简单 |
| Google Domains | 简洁易用 | 已迁移至Squarespace |

### 2.3 选择建议

- **国内用户**：选择阿里云或腾讯云，备案方便
- **海外项目**：选择Cloudflare或Namecheap
- **预算有限**：比较多家注册商价格
- **重视隐私**：选择提供免费隐私保护的注册商

## 三、域名购买流程

### 3.1 阿里云购买流程

#### 步骤1：账号准备

1. 访问 [阿里云官网](https://www.aliyun.com/)
2. 注册并登录账号
3. 完成实名认证

#### 步骤2：查询域名

1. 进入域名注册页面
2. 输入想要的域名
3. 查看是否可注册

#### 步骤3：购买域名

1. 选择可注册的域名
2. 添加到购物车
3. 选择购买年限（建议1-10年）
4. 选择信息模板（已完成实名）
5. 确认订单并支付

#### 步骤4：域名管理

```bash
# 在控制台可以：
- 查看域名列表
- 管理DNS解析
- 设置域名转出
- 续费域名
```

### 3.2 Cloudflare购买流程

#### 步骤1：账号准备

1. 访问 [Cloudflare官网](https://www.cloudflare.com/)
2. 注册并登录账号

#### 步骤2：购买域名

1. 进入 Domain Registration 页面
2. 搜索域名
3. 选择购买年限
4. 填写注册信息
5. 完成支付

#### 步骤3：自动配置

Cloudflare会自动：
- 添加域名到账户
- 配置DNS服务器
- 启用CDN代理

### 3.3 Namecheap购买流程

#### 步骤1：账号准备

1. 访问 [Namecheap官网](https://www.namecheap.com/)
2. 注册并登录账号

#### 步骤2：购买域名

1. 搜索域名
2. 添加到购物车
3. 选择购买年限
4. 启用WHOIS隐私保护（免费）
5. 结账支付

## 五、域名管理

### 5.1 域名解析

添加DNS解析记录：

| 记录类型 | 用途 | 示例 |
|---------|------|------|
| A | 指向IPv4地址 | @ → 192.168.1.1 |
| AAAA | 指向IPv6地址 | @ → 2001:db8::1 |
| CNAME | 指向另一个域名 | www → example.com |
| MX | 邮件服务器 | @ → mail.example.com |
| TXT | 文本记录 | SPF、验证等 |
| NS | 名称服务器 | 委托给其他DNS |

### 5.2 域名续费

```bash
# 建议设置
- 开启自动续费
- 设置续费提醒
- 账户保持足够余额
```

### 5.3 域名转移

#### 转出要求

- 域名注册超过60天
- 域名未过期
- 获取转移码（EPP码）

#### 转移流程

1. 在原注册商解锁域名
2. 获取转移码
3. 在新注册商提交转移申请
4. 确认转移邮件
5. 等待5-7天完成

## 六、域名安全

### 6.1 安全措施

- ✅ 启用域名锁定
- ✅ 开启两步验证
- ✅ 启用WHOIS隐私保护
- ✅ 使用强密码
- ✅ 定期检查账户活动

### 6.2 防止域名劫持

```bash
# 最佳实践
1. 使用企业邮箱注册（非个人邮箱）
2. 不要在公共场所登录账户
3. 定期更换密码
4. 关注域名到期通知
5. 保留注册证明文件
```

## 七、ICP备案（中国大陆）

### 7.1 备案要求

使用中国大陆服务器需要完成ICP备案：

- 域名实名认证
- 服务器在国内
- 网站内容合规

### 7.2 备案流程

1. 登录云服务商备案系统
2. 填写主体信息（个人/企业）
3. 填写网站信息
4. 上传资料（身份证、幕布照等）
5. 提交审核（约5-20个工作日）

### 7.3 备案号

备案成功后会获得备案号，需在网站底部显示：

```html
<a href="https://beian.miit.gov.cn/">
  京ICP备12345678号
</a>
```

## 八、常见问题

### Q1: 域名被注册怎么办？

- 尝试其他后缀（.net, .dev等）
- 添加前缀或后缀
- 使用缩写
- 联系域名持有者协商购买
- 等待域名过期（不推荐）

### Q2: 域名过期了怎么办？

```
域名过期时间线：
过期 → 宽限期(30天) → 赎回期(30天) → 删除期(5天) → 重新注册
       ↑ 高价赎回      ↑ 更高价赎回
```

建议：在宽限期内续费，费用较低

### Q3: 如何保护域名隐私？

大多数注册商提供WHOIS隐私保护服务：

```
未启用：
Registrant Name: 张三
Registrant Email: zhangsan@example.com

启用后：
Registrant Name: REDACTED FOR PRIVACY
Registrant Email: abc123@whoisprivacy.com
```

### Q4: 子域名需要单独购买吗？

不需要。购买域名后可以创建无限子域名：

```
example.com        ← 购买的域名
www.example.com    ← 子域名
blog.example.com   ← 子域名
api.example.com    ← 子域名
```
