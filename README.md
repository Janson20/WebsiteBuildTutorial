# 网站搭建完整教程

从零开始搭建网站的完整指南，涵盖代码编写、服务器部署、域名购买和DNS配置。

## 教程目录

### 第一部分：代码开发
- [01-code-development.md](./01-code-development.md) - 代码编写最佳实践与项目结构

### 第二部分：服务器部署
- [02-server-deployment.md](./02-server-deployment.md) - 服务器选择、配置与项目部署

### 第三部分：域名管理
- [03-domain-purchase.md](./03-domain-purchase.md) - 域名购买与注册指南

### 第四部分：DNS配置
- [04-cloudflare-dns.md](./04-cloudflare-dns.md) - Cloudflare DNS设置与优化

### 第五部分：完整流程
- [05-complete-workflow.md](./05-complete-workflow.md) - 从零到上线的完整实战案例

## 教程特色

- **循序渐进**：从基础到进阶，适合不同水平的学习者
- **实用导向**：每一步都有具体的操作指南
- **问题解决**：包含常见问题的排查与解决方案
- **最佳实践**：总结业界推荐的配置和安全措施

## 目标读者

- 想要搭建个人博客或作品集的开发者
- 需要部署Web应用的前端/全栈工程师
- 对网站运维感兴趣的技术爱好者

## 前置知识

- 基本的HTML/CSS/JavaScript知识
- 了解Git版本控制基本操作
- 熟悉命令行基本命令

## 技术栈

本教程涵盖以下技术：

| 类别 | 技术 |
|------|------|
| 前端框架 | React, Vue, Next.js, Nuxt.js |
| 静态站点 | Hugo, Hexo, VitePress |
| 服务器 | Linux (Ubuntu/CentOS), Nginx |
| 云服务 | 阿里云, 腾讯云, AWS, Vercel, Netlify |
| CDN/DNS | Cloudflare |
